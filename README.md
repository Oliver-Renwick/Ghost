# Ghost

**Ghost** is a game engine built with the **Win32 API** and **Vulkan API**. It features an **impulse-based physics engine** with **collision detection** and **shadow rendering**, offering realistic physics simulations and dynamic lighting for interactive environments. The engine is currently **integrating Client - Server functionality** using **Winsock**, aiming to support real-time networked gameplay.


![ghost Screenshot 2025 02 12 - 21 43 03 21](https://github.com/user-attachments/assets/f3737a83-a1ab-4c3d-8455-6fc2e20574d2)


## Features
-  Vulkan-based rendering
- Impulse-based physics engine
- Collision detection
- Dynamic shadow rendering
- Multiplayer integration in progress using Winsock


## Build

### Prerequisites
- **Visual Studio 2022**  
- **Vulkan API**  

### Building on Windows
Ensure all prerequisites are installed. To generate the project files, run:

```batch
Setup.bat
```

> **Note**  
> This engine is built exclusively using the Win32 API and is supported only on Windows.
